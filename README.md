# BMI Calculator 

A simple Python GUI application that calculates Body Mass Index (BMI) based on user input. Just enter your weight in kilograms and height in feet, and the app will display your BMI along with a health category (Underweight, Normal, Overweight, etc.).

## Features
- Input weight in kilograms
- Input height in feet
- Calculates BMI instantly
- Displays health status based on BMI
- Easy-to-use graphical interface

## Technologies Used
- Python
- Tkinter (for GUI)
